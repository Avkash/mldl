## Python Server (one line)
- On python 3
 - $ python -m http.server
 - Note: default port will be 8000, to run server on desired port do the following:
 - $ python -m http.server 8080
- On python 2
 - $ python -m SimpleHTTPServer
 - Note: default port will be 8000, to run server on desired port do the following:
 - $ python -m SimpleHTTPServer 8888
